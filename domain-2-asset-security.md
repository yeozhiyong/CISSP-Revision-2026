## 🎴 Flashcards - Data Roles

| Front | Back |
|-------|------|
| **Data Owner** | **Classifies** data, determines **value**, **approves** access, **business person** |
| **Data Custodian** | **Implements** controls, manages **storage**, **IT person** |
| **Data User** | **Follows** rules, uses data **appropriately**, **employee** |

**Memory Trick:**
- **Owner** = **O**verall responsibility = **Decides**
- **Custodian** = **C**ares for data = **Does the work**

---

## 🎴 Flashcards - Data States

| Front | Back |
|-------|------|
| **Data at Rest** | In **storage** (disk, database, backup) → Encrypt |
| **Data in Transit** | On **network** (LAN, internet) → TLS, VPN |
| **Data in Use** | In **memory/CPU** (active processing) → Secure enclaves |

---

## 🎴 Flashcards - Classification Levels

| Level | Description | Example |
|-------|-------------|---------|
| **Public** | No harm if disclosed | Marketing website |
| **Internal** | Business use only | Phone directory, org charts |
| **Confidential** | Significant harm if disclosed | Customer data, financials |
| **Secret/Restricted** | Severe harm if disclosed | Trade secrets, M&A plans |

**Key Point:** Classification based on **business impact**, not technical difficulty!

## 🎴 Flashcards - Secure Disposal

| Media | Method | Why |
|-------|--------|-----|
| **HDD** | Degaussing ✅ | Magnetic storage |
| **SSD** | **Physical destruction** ✅ | Flash memory, NOT magnetic |
| **SSD** | Degaussing ❌ | **Doesn't work!** |
| **Paper** | Shredding, burning | Physical destruction |

**CISSP Exam Trap:** Deleting or formatting is **NOT** secure disposal!
