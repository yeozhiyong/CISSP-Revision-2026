🔬 Universal Research Workflow

Multi-Source Research Automation by Cupid

📋 Overview

The Universal Research Workflow automates comprehensive research across multiple sources, synthesizes findings with AI, and delivers structured reports to GitHub, Notion, and Slack.

Tavily API YouTube API GitHub API Notion API Slack API Kimi K2.5

🔄 Workflow Steps

1

🎯 Input Query

User provides research topic via command line

./universal-research.sh "LLM comparison enterprise 2026"
↓
2

🔍 Parallel Data Collection

Simultaneous search across multiple sources:

• Tavily Web Search: 10+ web results with AI summaries
• YouTube Search: 5 relevant videos
• GitHub Search: Top repositories
↓
3

🤖 AI Synthesis

Kimi K2.5 analyzes and synthesizes all sources:

• Extracts key insights
• Identifies trends and patterns
• Structures findings into report format
↓
4

📝 Report Generation

Creates comprehensive markdown report:

• Executive Summary
• Key Findings
• Source Analysis
• Recommendations
↓
5

API Keys Required

  • TAVILY_API_KEY - Web search
  • YOUTUBE_API_KEY - Video search
  • GITHUB_TOKEN - Repository access
  • NOTION_TOKEN - Database integration
  • SLACK_BOT_TOKEN - Notifications
